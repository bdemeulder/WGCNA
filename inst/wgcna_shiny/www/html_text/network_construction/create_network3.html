The user can download the modules assignments for all genes in the analysis by clicking on the Download Module Assignments button. The user can also download the HTML report of this page by clicking on the Download Report button.

A useful check for module definitions can be performed by verifying that there is a grey module. The size of the grey module should be related to the amount of noise expected.

Once the user is happy with the definition of modules, he/she can go to the third part of the program by clicking on the Go to Next Step button, or the Module – Trait Relationship tab at the top of the page.
